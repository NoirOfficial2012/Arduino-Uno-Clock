# Arduino-Uno-Clock
I made this project using just arduino uno and a display! To prove that its super Easy to make a clock.
Its super easy to DIY 
change this part of the code to your time and date accordingly:

// Time + date
int hours = 14, minutes = 11, seconds = 50;
int day = 2, month = 4, year = 2026;
The Parts Used In the project are 
1.Arduino UNO
2.TFT SHEILD display For arduino uno
There is no need to do complicated wiring in the hardware as the tft sheild display is made for arduino uno it just snaps right on it :)
